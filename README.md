🧩 <b>To-Do List App (React)</b>

Aplicación de gestión de tareas desarrollada con React, que permite crear, editar, eliminar y organizar tareas con persistencia de datos en el navegador.

🚀 <b>Demo</b>

👉 https://jonathangalimberti.github.io/curso-react-intro/

📦 <b>Repositorio</b>

👉 https://github.com/jonathangalimberti/curso-react-intro.git

✨ <b>Features</b><br/>
✔ Crear tareas<br/>
✔ Editar tareas<br/>
✔ Eliminar tareas<br/>
✔ Marcar como completadas<br/>
✔ Filtros (todas / pendientes / completadas)<br/>
✔ Persistencia con localStorage<br/>
✔ Interfaz dinámica<br/>

🛠️ <b>Tecnologías utilizadas</b><br/>
JavaScript (ES6+)<br/>
React<br/>
HTML5<br/>
CSS3<br/>
localStorage<br/>

🧠 <b>Conceptos aplicados </b>

Este proyecto me permitió practicar:

Manejo de estado con useState<br/>
Efectos con useEffect<br/>
Inmutabilidad del estado<br/>
Renderizado dinámico<br/>
Manejo de eventos<br/>
Persistencia de datos en el navegador<br/>

⚙️<b> Instalación local</b>

Si querés correr el proyecto en tu máquina:

git clone https://github.com/jonathangalimberti/curso-react-intro.git<br/>
cd curso-react-intro<br/>
npm install<br/>
npm start
